# Shogi
## Insruction
>This Assignment is to use C to make a shogi game for two real players to play chess. 


## Compile
### Download libev
    sudo apt-get update
    sudo apt-get install libev-dev
### compile
    cd assignment_2-shogi-Bubuchance
    cd src
    gcc hw2.c -lev -o hw2
## Run
### Open a new game
    ./hw2 [-n][-s newfile]
### Reload old game file
    ./hw2 [-l oldfile]
